
# 🛒 Bestellapp

A responsive food ordering app built with HTML, CSS, and JavaScript. Users can add dishes to the cart, update quantities, switch between delivery and pickup, and place orders with instant feedback.

## ✨ Features

<ul> 
  <li>📱 Responsive design</li> 
  <li>🛒 Add items to the cart</li> 
  <li>➕➖ Update item quantities</li> 
  <li>❌ Remove items from the cart</li> 
  <li>💰 Automatic total price calculation</li> 
  <li>🚚 Switch between delivery & pickup</li> 
  <li>✅ Order confirmation feedback</li>
  <li>⚡ Dynamic cart updates</li>

## 💡 Lessons Learned

This project helped me gain a deeper understanding of working with JSON data and dynamically rendering content in JavaScript. I learned how to navigate through JSON objects and arrays using loops to retrieve and display information such as dish names and prices.

Additionally, I learned how to create and reuse HTML templates, making the code more modular, maintainable, and easier to extend. This project also strengthened my JavaScript skills by building dynamic user interfaces and reinforced the importance of writing clean, reusable code.
